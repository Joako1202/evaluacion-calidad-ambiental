# Automatización de Evaluación de Resultados de Calidad de Suelo

## Descripción
Este proyecto contiene un script en Python que **automatiza la evaluación de resultados de calidad de suelo** para cumplir con el programa de monitoreo de un proyecto de inversión. Permite:

- Identificar resultados que **superan los ECA de Suelo 2017- D.S. 011-2017-MINAM**.
- Realizar **análisis estadístico**: máximos, mínimos, promedios.
- Determinar en qué **estaciones y fechas** se registraron las excedencias.
- Generar **gráficos automáticamente**.

El procesamiento completo toma **aproximadamente 4 minutos**, comparado con **3 horas de análisis manual**.

## Dependencias
El script utiliza las siguientes librerías de Python:

- **Pandas** → manejo y análisis de datos tabulares (CSV, Excel, DataFrames)
- **NumPy** → operaciones numéricas y lógicas vectorizadas
- **Matplotlib** → generación de gráficos y visualizaciones


El script generará:
Tablas de resultados que superan los ECA
Estadísticas básicas (máximos, mínimos, promedios)
Gráficos por estación y fecha
Archivos de salida en output/
